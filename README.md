# Rock Paper Scissors 🎮

My first Python project: a simple Rock Paper Scissors game played against the computer.

## Features

- Choose Rock, Paper, or Scissors
- Random computer choice
- Winner determination
- Tie detection
- Emoji-based interface for a cleaner user experience

## How to Run

1. Make sure Python is installed.
2. Download or clone this repository.
3. Run the program:

```bash
python rock_paper_scissors.py
```

## Example Output

```text

==============
```
